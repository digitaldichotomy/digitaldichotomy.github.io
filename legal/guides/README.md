# Legal Guides Documentation

## Purpose
This folder contains guides for legal actions, class actions, and advocacy strategies.

## How to Contribute
1. Add new guides as `guide-[description].md`.
2. Include step-by-step instructions.
3. Use the template:
   ```markdown
   ---
   title: "[Guide Name]"
   ---
   ### Summary
   [Description]
   ```
