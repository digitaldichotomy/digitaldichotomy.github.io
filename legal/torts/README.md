# Digital Battery Tort Documentation

## Purpose
This folder contains legal frameworks and patent analyses for **Digital Battery Tort**, a new legal theory defining non-consensual biofield contact as a tortious act.

## How to Contribute
1. Add new tort analyses as `tort-[description].md`.
2. Link to relevant patents in `/evidence/patents/`.
3. Use the template:
   ```markdown
   ---
   title: "[Tort Name]"
   patents:
     - US1234567
     - US8910111
   ---
   ### Summary
   [Description]
   ```
