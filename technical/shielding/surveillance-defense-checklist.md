# Surveillance Defense Checklist

## Daily Practices (Complete Every Day)

### Morning Routine
□ Cover phone cameras when not in use
□ Disable location services on mobile devices
□ Check for unusual device behavior
□ Review app permissions for new apps

### During Day
□ Use encrypted messaging apps (Signal, Session)
□ Avoid public WiFi for sensitive communications
□ Mix up your daily routes and timing
□ Use cash for privacy-sensitive purchases

### Evening Routine  
□ Clear browser history and cookies
□ Review social media privacy settings
□ Check for unknown devices on network
□ Backup important data to encrypted storage

## Weekly Security Tasks

### Digital Security
□ Update all software and operating systems

□ Check email forwarding and filters
□ Audit social media friend/follower lists

### Physical Security
□ Sweep home for hidden cameras/mics
□ Test RF shielding effectiveness
□ Check vehicle for tracking devices
□ Review home network security logs

### Communication Security
□ Rotate VPN servers/locations
□ Update encryption keys
□ Review secure messaging contacts
□ Practice operational security (OPSEC)

## Monthly Deep Security

### Data Management

### Network Security
□ Change WiFi passwords
□ Update router firmware
□ Check network for unknown connections

### Physical Environment

## Emergency Response Plan

### If Compromised
□ Disconnect from all networks immediately
□ Remove batteries from electronic devices
□ Move to secure location
□ Contact trusted security advisor

### If Under Active Attack
□ Document all symptoms and evidence
□ Seek medical attention if needed
□ Contact law enforcement if appropriate
□ Reach out to digital rights organizations
