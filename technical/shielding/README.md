# Technical Shielding Documentation

## Purpose
This folder contains manuals and checklists for electromagnetic shielding and surveillance defense.

## How to Contribute
1. Add new shielding docs as `shielding-[description].md`.
2. Include practical guides.
3. Use the template:
   ```markdown
   ---
   title: "[Shielding Name]"
   ---
   ### Summary
   [Description]
   ```
