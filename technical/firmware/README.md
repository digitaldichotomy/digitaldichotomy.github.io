# Technical Firmware Documentation

## Purpose
This folder contains analyses of firmware and core technical components related to surveillance and countermeasures.

## How to Contribute
1. Add new firmware docs as `firmware-[description].md`.
2. Include technical details and references.
3. Use the template:
   ```markdown
   ---
   title: "[Firmware Name]"
   ---
   ### Summary
   [Description]
   ```
