## Platform-Specific Adjustments

### TikTok
- Use all 8 hashtags
- Put in caption (not comments)
- First hashtag = most important for algorithm

### Instagram Reels
- Use 8-12 hashtags
- Mix in 2-3 trending tags if relevant
- Put in caption OR first comment (test both)

### YouTube Shorts
- Use 3-5 hashtags max
- Include #shorts always
- Put in title + description
