### Indiana Biometric Privacy Act (IBPA) {#indiana-biometric-privacy-act-ibpa .text-2xl .font-bold .text-purple-300 .mb-4}

::: text-gray-300
Indiana\'s IBPA provides strong protections for biometric data privacy:

::: {.bg-purple-900/30 .border .border-purple-700 .rounded-lg .p-4 .mb-4}
#### Key Provisions: {#key-provisions .font-bold .text-purple-200 .mb-2}

-   ‚Ä¢ Requires informed consent before collecting biometric data
-   ‚Ä¢ Prohibits sale of biometric information without consent
-   ‚Ä¢ Allows for damages of \$1,000 per negligent violation
-   ‚Ä¢ Provides for \$5,000 per reckless/intentional violation
-   ‚Ä¢ Provides for attorney fees and costs
:::

Apple\'s sensor fusion systems violate the IBPA by:

1.  Collecting motion, touch, and usage data to infer
    emotional/cognitive states
2.  Processing this data to create biometric identifiers without consent
3.  Failed to disclose the nature and purpose of biometric data
    collection
4.  Transmitting inferred biometric data to third-party servers
:::
:::

::: {.bg-gray-900/60 .border .border-gray-700 .rounded-lg .p-6}
### Common Law Invasion of Privacy {#common-law-invasion-of-privacy .text-2xl .font-bold .text-green-300 .mb-4}

::: text-gray-300
Indiana recognizes invasion of privacy claims for technological
violations:

::: {.bg-green-900/30 .border .border-green-700 .rounded-lg .p-4}
#### Privacy Torts in Indiana: {#privacy-torts-in-indiana .font-bold .text-green-200 .mb-2}

-   \*\*Intrusion upon solitude\*\*: Unauthorized collection of private
    biometric data
-   \*\*Public disclosure\*\*: Revealing private biometric
    characteristics
-   \*\*False light\*\*: Misrepresenting the nature of data collection
-   \*\*Appropriation\*\*: Using biometric characteristics for
    commercial purposes
:::

Indiana courts have extended privacy protections to cover emerging
technologies that intrude on personal autonomy.
