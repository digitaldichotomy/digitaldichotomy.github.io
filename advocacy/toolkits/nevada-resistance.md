-   ‚Ä¢ REG 6.010 requires disclosure of surveillance technologies to
    patrons
-   ‚Ä¢ REG 5.110 prohibits using surveillance for non-gaming purposes
-   ‚Ä¢ REG 3.040 protects patron privacy rights on casino premises
-   ‚Ä¢ Nevada Gaming Board: (702) 486-2000 ‚Ä¢ Complaint line: (702)
    486-2020

### Nevada Biometric Privacy Law (SB220) {#nevada-biometric-privacy-law-sb220 .text-2xl .font-bold .text-green-300 .mb-4}

::: text-gray-300
Nevada\'s biometric privacy law provides strong protections:

::: {.bg-green-900/30 .border .border-green-700 .rounded-lg .p-4 .mb-4}
#### Key Provisions: {#key-provisions .font-bold .text-green-200 .mb-2}

-   ‚Ä¢ Requires consent before collecting biometric data
-   ‚Ä¢ Prohibits sale of biometric information without consent
-   ‚Ä¢ Allows for actual damages and attorney fees
-   ‚Ä¢ Applies to facial recognition, voice analysis, emotional detection
:::

Google\'s voice analysis programs violate SB220 by:

1.  Analyzing voice stress during YouTube viewing
2.  Inferring emotional state without explicit consent
3.  Using inferred states to target gambling advertisements
4.  Failure to disclose emotional state analysis to users
:::
:::

::: {.bg-gray-900/60 .border .border-gray-700 .rounded-lg .p-6}

::: {.bg-blue-900/30 .border .border-blue-700 .rounded-lg .p-4}
#### Prohibited Practices: {#prohibited-practices .font-bold .text-blue-200 .mb-2}

-   ‚Ä¢ Deceptive acts or practices
-   ‚Ä¢ Unconscionable acts or practices
-   ‚Ä¢ Failure to disclose material information
-   ‚Ä¢ Targeting vulnerable populations for exploitation

Targeting gambling addicts with emotionally manipulative ads violates
