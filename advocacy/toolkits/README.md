# Advocacy Toolkits Documentation

## Purpose
This folder contains toolkits for activism, resistance, and press outreach.

## How to Contribute
1. Add new toolkits as `toolkit-[description].md`.
2. Include actionable steps.
3. Use the template:
   ```markdown
   ---
   title: "[Toolkit Name]"
   ---
   ### Summary
   [Description]
   ```
