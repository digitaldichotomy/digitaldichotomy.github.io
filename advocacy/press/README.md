# Advocacy Press Documentation

## Purpose
This folder contains press releases and media templates for advocacy.

## How to Contribute
1. Add new press items as `press-[description].md`.
2. Use professional templates.
3. Template:
   ```markdown
   ---
   title: "[Press Release Title]"
   ---
   ### Summary
   [Description]
   ```
