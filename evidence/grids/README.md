# Evidence Grids Documentation

## Purpose
This folder contains grids and mappings of biometric and surveillance technologies.

## How to Contribute
1. Add new grids as `grid-[description].md`.
2. Include diagrams and references.
3. Use the template:
   ```markdown
   ---
   title: "[Grid Name]"
   ---
   ### Summary
   [Description]
   ```
