# Evidence Patents Documentation

## Purpose
This folder contains patent analyses and databases for surveillance technologies.

## How to Contribute
1. Add new patents as `tier-[number].md`.
2. Include patent numbers and summaries.
3. Use the template:
   ```markdown
   ---
   title: "[Patent Name]"
   tier: [Number]
   ---
   ### Summary
   [Description]
   ```
