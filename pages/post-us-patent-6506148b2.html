1. Executive Summary

This patent describes a method and apparatus for manipulating the human nervous system by pulsing the intensity of images displayed on a standard computer monitor or television. The core principle is that these pulses create weak electromagnetic (EM) fields at specific frequencies. When a person is exposed to these fields, their cutaneous (skin) nerves can be stimulated, exciting what the patent calls "sensory resonances" in the brain. This can induce various physiological and psychological effects, ranging from drowsiness and relaxation to specific physical sensations. A key aspect of the invention is that the image pulsing can be subliminal—too subtle to be consciously perceived by the viewer—yet still be effective.

2. The Core Concept: Sensory Resonance

The patent is built on the theory that the human nervous system has specific resonance frequencies. By stimulating sensory pathways (in this case, nerves in the skin) with a signal pulsed at one of these frequencies, the spontaneous firing patterns of the nerves can be modulated. This modulated signal is then conveyed to the brain.

The patent highlights two primary resonance frequencies:

  • ½ Hz (0.5 Hz): Associated with effects like ptosis (drooping of the eyelids), relaxation, drowsiness, a tonic smile, and a "tense feeling in the stomach."
  • 2.4 Hz: Associated with the slowing of certain cortical processes.

3. Methods of Implementation

The patent details several ways to create the required pulsed EM fields using standard display technology. The common goal is to modulate the luminance (intensity) of the displayed image at the desired resonance frequency.

3.1. Hardware Modulation for Video Sources

For video sources like a VCR or DVD player, a hardware modulator can be inserted between the source and the monitor. This device adds a low-frequency pulse to the video signal's luminance component without affecting the color (chrominance), causing the overall image intensity to throb rhythmically.

3.2. Software Control for Computer Monitors

On a computer, a program can be used to directly control the display adapter. The program rhythmically adjusts the color values (RGB) of the entire screen or a portion of it by a small percentage. The patent describes a Visual Basic program that uses precise timing to generate a smooth, sine-wave-like pulse at the target frequency.

Figure 4: A setup where a computer program controls the monitor to generate pulsed images. The subject can adjust the frequency and amplitude via the keyboard.

3.3. Remote and Internet-Based Operation

The patent explicitly states that the controlling software does not need to be on the user's local machine. The program could be run on a remote server and delivered via a network, such as the Internet.

Figure 9: An illustration of the system operating over a network. The image-pulsing program runs on a remote computer, affecting the monitor connected to the user's computer via a link like the Internet.

4. Software Logic and Timing

To achieve the precise low-frequency pulses (e.g., 0.5 Hz) required for sensory resonance, the patent outlines a specific software algorithm. Since standard system timers can be inaccurate, the program uses a high-resolution system timer (GetTickCount) and an extrapolation method to ensure smooth and accurate pulsing.

5. The Physics of Monitor Emission

The patent explains the physical mechanism by which the pulsed image creates an EM field.

  • CRT Monitors: The electron beam that draws the image is a current. Pulsing the image intensity pulses this current. The accumulation of charge on the screen and the current itself generate both electric and magnetic fields that radiate outwards.
  • LCD Monitors: The varying voltage potentials on the display's electrode plates, which control the liquid crystals, create a pulsed electric field that is analogous to the field from a capacitor.

6. Subliminal Operation and Implications

A significant claim of the patent is that the pulsing can be made subliminal. By modulating the image intensity by only a small fraction (e.g., 1%), the visual change is imperceptible to the conscious mind. However, the patent asserts that this is still sufficient to generate an EM field strong enough to excite the sensory resonances. The inventor notes this creates a potential for "mischievous application," where individuals could be exposed to nervous system manipulation without their knowledge while watching TV or using a computer.

7. Conclusion

U.S. Patent 6506148B2 provides a detailed technical blueprint for a system designed to affect the human nervous system using the electromagnetic byproducts of common display devices. It combines physiological theories of "sensory resonance" with practical engineering solutions for generating precise, low-frequency EM pulses through both hardware and software. The patent expired in 2021 and is now in the public domain.
